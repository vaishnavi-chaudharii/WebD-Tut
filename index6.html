Welcome Future Developers

1. HTML: HTML stands for Hypertext Markup Language. It is the language that web developers use to create websites. HTML is what makes a website look nice and professional. HTML is the foundation of any website. If you want to learn how to make a website, then you need to know HTML first. You can learn HTML online at sites like Codecademy.com or Udacity.com. 2. CSS: CSS stands for Cascading Style Sheets. A style sheet is a way to apply formatting to text and images on a webpage. There are different types of styles, including font styles (bold, italics), colors, background colors, alignment, margins, etc. CSS is used to add these formatting attributes to a page. To learn more about CSS, check out sites like W3Schools.com. 3. JavaScript: JavaScript is a scripting language that adds interactivity to a web page. When a user clicks on something on a web page, JavaScript tells the browser what to do. JavaScript is often used to add special effects to a web page, like making a button roll down a window. Learn more about JavaScript at sites like CodeAcademy.com.
